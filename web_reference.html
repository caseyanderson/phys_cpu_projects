Arduino, start here for all Arduino-related information. StackOverflow, probably the best question/answer platform for development work in existence. Electronics.StackExchange, the same system as the above, but only for Electronics. Adafruit Learning System, Adafruit's collection of tutorials for things they sell. Learn at Sparkfun, SparkFun's collection of tutorials for things they sell. Open Electronics, nice collection of open source hardware projects (complete with schematic and tutorials). Dive Into Python, my favorite book on learning to code with Python. Linux Bash Shell cheat sheet, until you start memorizing these (you will eventually memorize some of them through shear repetition, I promise), it is nice to have a cheat sheet on hand.
