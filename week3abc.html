LAB: Version Control with Github

All of the materials for this class live in two places: this website, and this GitHub repository. Following the completion of this lab, it will be your responsibility to keep up to date with the classroom materials as they are added.
Start by signing up for a GitHub account (it's free) here.
Next, go here to download and install the latest version of Git.
At the terminal, tell git your username by entering git config --global user.name "USERNAME".
Then tell git your email address (the one associated with your GitHub account) by entering git config --global user.email "YOUR EMAIL ADDRESS".
To confirm that you have configured git correctly, check the .gitconfig file by running the following line at the terminal: sudo nano .gitconfig.
In the next sequence, we are going to move inside of the NEWDIRECTORY folder we made in the first lab and clone the class repository: 
cd NEWDIRECTORY
git clone https://github.com/caseyanderson/phys_cpu_projects.git
#git clone copies the repository into a new directory in this folder
cd phys_cpu_projects
git log # shows the history of this repository
# type q to exit log
git pull # check for updates from source
# if there are updates, download them

At least once a week you should git pull inside this directory to stay up to date.
